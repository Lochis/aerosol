# Aerosol

## Usage

### With Node.js

1. Install dependencies:

   ```sh
   npm install
   ```

2. Run client and server concurrently in development mode:

   ```sh
   npm run dev --workspaces
   ```

3. Build client and server concurrently:

   ```sh
   npm run build --workspaces
   ```

### With Bun

1. Install dependencies:

   ```sh
   bun install
   ```

2. Run client and server concurrently in development mode:

   ```sh
   bun dev --workspaces
   ```

3. Build client and server concurrently:

   ```sh
   bun build --workspaces
   ```
