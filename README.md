# Aerosol

## Usage

1. Install Bun: https://bun.sh/

2. Install dependencies using Bun:

   ```sh
   bun install
   ```

3. Run client and server in development mode:

   ```sh
   bun dev
   ```

4. Build the client and server files (note the extra `run` keyword):

   ```sh
   bun run build
   ```
